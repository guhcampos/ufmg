INTRODUÇÃO

Defend the Cake é um jogo no já tradicional gênero de Tower Defense. Nesse tipo de jogo, o jogador deve colocar no mapa "torres" com o objetivo de impedir hordas de inimigos de atingir um determinado objetivo. Os inimigos podem caminhar por rotas predefinidas ou, nos jogos mais elaborados, adaptar suas rotas dinamicamente em resposta às alterações de cenário criadas pela colocação de torres.

Em "Defend the Cake", você deve defender o seu bolo de aniversário de uma horda de formigas sedentas por açucar. Para isso, você dispõe apenas de torres providas de canhões.

OBJETIVO

O jogador pode colocar torres nas bordas do caminho das formigas, nos quadros marcados com o montinho de terra. Colocar uma torre custa 5 pontos, o que obriga o jogador a conseguir eliminar 5 formigas para colocar a segunda torre, e assim por diante. O objetivo do jogo é, portanto, maximizar o score utilizando o menor número de torres possível para eliminar todas as formigas. São três fases com nível de dificuldade crescente: o caminho das formigas fica mais curto, e as opções para colocação de torres mais restritas. Além disso, em cada mapa, matar uma certa quantidade de formigas irá despertar nelas um modo "bersek", e elas passarão a se mover com velocidade alucinante em direção ao delicioso bolo.

CONTROLES

Para colocar torres, basta clicar no montinho de terra desejado, dispondo dos 5 pontos necessários para colocar uma nova torre. O jogador deverá ser automaticamente transportado entre as fases quando a horda de formigas em cada fase acabar. Se alguma formiga tocar no bolo, tudo estará perdido e as formigas comemorarão sua vitória açucarada.

CHEATS

O jogo é relativamente difícil e, apesar de simples, exige uma coordenação entre estratégia e agilidade. Para tornar as coisas mais simples, existem dois "cheats" disponíveis:

1. Apertando a barra de espaço, o jogador será transferido automaticamente para a próxima fase.
2. Segurando os botões de seta para cima e para baixo, o jogador pode ajustar a frequência de fogo das torres. Utilizando a frequência máxima o jogador deverá ser capaz de ganhar qualquer fase com uma única torre. Da mesma forma, se quiser aumentar o desafio, o botão seta para baixo fará com que as torres atirem mais devagar, facilitando a vida das formigas.
