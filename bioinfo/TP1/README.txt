Olá =)

Para rodar o programa, deve ser suficiente invocar tp.sh.

Trata-se de um script BASH que executa o programa com os
argumentos corretos. Há algumas linhas comentadas nesse
script, que se descomentadas (e comentada a linha que é
executada no momento) invocam uma versão "light" do
trabalho, que roda em alguns poucos segundos.

O programa só possui um requisito além da biblioteca padrão
do Python 2.7.5: o pacote numpy.

Esse pacote foi utilizado para a criação de matrizes rápidas
para representação de números inteiros, já que sua implementação
de matrizes é consideravelmente mais rápida que o usual
"listas de listas" usado em Python tradicional
