Arquivos:


tp3_geragrafo.py: cria grafos de proteínas usando arquivos PDB diretamente

tp3_geragrafo_gzip.py: cria grafos de proteínas usando arquivos PDB gzipados

geragrafos.sh: script para gerar automaticamente todos os grafos da família
